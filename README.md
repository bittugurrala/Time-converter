
# Time Converter

A simple and responsive **Time Converter** web app built using **HTML**, **CSS**, and **JavaScript**.  
It allows users to convert hours and minutes into seconds with instant validation and error feedback.

---

##  Features

- Convert **hours** and **minutes** into **seconds**  
- Validation for empty or negative inputs  
- Responsive UI styled with **Bootstrap 5**  
- Inline error messages for missing fields  
- Custom color theme (`#867BE2` as the primary base)

---

##  Tech Stack

| Technology | Purpose |
|-------------|----------|
| **HTML5** | Structure and content |
| **CSS3 / Bootstrap 5** | Styling and layout |
| **JavaScript (ES6)** | Input validation and calculation logic |

## How It Works

1. Enter the number of **hours** and **minutes**.
2. Click the **Convert into seconds!** button.
3. The total time in seconds is displayed instantly.
4. Error alerts appear if:
   - Fields are empty
   - Negative values are entered
